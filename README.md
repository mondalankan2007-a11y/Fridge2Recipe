Project Title:
Fridge2Recipe – Find Meals From Ingredients You Have

Project Description:
Fridge2Recipe is a website that assists users in finding recipes using the ingredients they already have at home. Users can input ingredients they have on hand in their refrigerator, and the app will show meals that can be made with those items.

The app retrieves up-to-date recipe information from a public API and displays it in a clear, easy-to-use interface. Each recipe provides cooking steps, a list of ingredients, and extra information to assist users in preparing meals with ease.

This project showcases essential JavaScript concepts, including API integration, dynamic updates to the user interface, and the application of array higher-order functions for searching, filtering, and sorting.
