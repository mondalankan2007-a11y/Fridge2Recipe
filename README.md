# 🍽️ Meal Master – Recipe Finder

**Meal Master** is a simple and interactive recipe finder web application that helps users discover meals they can cook using ingredients already available in their fridge. The application fetches recipe data from **TheMealDB API** and displays meal images, ingredients, and cooking instructions in an easy-to-use interface.

---

## 🚀 Features
- 🔍 **Search Recipes** – Find meals using ingredients or meal names  
- 🍲 **Recipe Details** – View meal image, ingredients, and cooking instructions  
- 🧾 **Dynamic Ingredient List** – Ingredients are displayed using JavaScript mapping  
- 🔗 **Recipe Pages with URL Parameters** – Each recipe can be viewed separately  
- 📊 **Nutrition Facts Toggle** – Option to show or hide nutrition details  
- 🛒 **Shopping List Generator** – Combine ingredients from **3 different recipes**

---

## 🛠️ Technologies Used
- **HTML5** – Structure of the application  
- **CSS3** – Styling and layout  
- **JavaScript (Vanilla JS)** – API calls and dynamic content  
- **TheMealDB API** – Recipe data source  

---

## 📡 API Used

Search meal by name  https://www.themealdb.com/api/json/v1/1/search.php?s=Arrabiata

List meals by first letter  https://www.themealdb.com/api/json/v1/1/search.php?f=a



API Documentation: https://www.themealdb.com/api.php

---

## ⚙️ How It Works
1. User enters an ingredient or meal name.  
2. JavaScript sends a request to **TheMealDB API**.  
3. The API returns recipe data in **JSON format**.  
4. The app dynamically displays meals, ingredients, and cooking instructions.
