🍽️ Meal Master – Recipe Finder

Meal Master is a simple and interactive recipe search web application that helps users find meals they can cook using ingredients already available in their fridge.

The app fetches recipe data from the TheMealDB API and displays meal details including ingredients, cooking instructions, and images.

This project is built using HTML, CSS, and JavaScript.

🚀 Features
🔍 Search Recipes by Ingredient
Enter ingredients you have in your fridge and discover meals you can cook.
🍲 Recipe Details Page
View complete recipe information including:
Ingredients
Cooking instructions
Meal image
🧾 Ingredient Mapping
Ingredients are dynamically displayed using JavaScript loops.
🔗 URL Parameters for Recipe Pages
Each recipe has its own view page using URL parameters.
🧮 Nutrition Facts Toggle
Users can toggle nutrition information for the selected recipe.
🛒 Shopping List Feature
Combine ingredients from three different recipes into a single shopping list.
🛠️ Technologies Used
HTML5 – Structure of the application
CSS3 – Styling and layout
JavaScript (Vanilla JS) – API calls and dynamic content
TheMealDB API – Recipe data source
📡 API Used

This project uses the TheMealDB API.

Example endpoints:

Search meal by name

https://www.themealdb.com/api/json/v1/1/search.php?s=Arrabiata

List meals by first letter

https://www.themealdb.com/api/json/v1/1/search.php?f=a

Official API documentation:
https://www.themealdb.com/api.php

🍽️ Meal Master – Recipe Finder

Meal Master is a simple and interactive web application that helps users discover recipes using ingredients they already have in their fridge.

Instead of searching randomly for recipes, users can enter available ingredients and quickly find meals they can cook. The application fetches recipe data from TheMealDB API and displays detailed cooking instructions, ingredients, and meal images.

🚀 Live Features

🔍 Search Recipes
Find meals by entering an ingredient or meal name.

🍲 Recipe Details Page
Each recipe has its own page where users can see:

Meal image
Ingredients list
Cooking instructions

🧾 Dynamic Ingredient Mapping
Ingredients are automatically displayed using JavaScript loops.

🔗 URL-based Recipe Pages
Recipe details are shown using URL parameters so every meal has its own link.

📊 Nutrition Facts Toggle
Users can toggle nutrition information for recipes.

🛒 Shopping List Generator
Users can select 3 recipes and combine their ingredients into a single shopping list.
